# TP1 Flutter — Interfaces Graphiques Mobiles

Ce dépôt contient la réalisation du **TP1 Flutter**, composé de deux exercices destinés à découvrir la création d'interfaces graphiques mobiles, la gestion d’état et l’architecture d’une application Flutter.

---

## Exercice 1 — Profile Card

L’objectif était de créer une interface statique affichant une carte de profil :

- Photo de profil (avatar)
- Carte stylisée avec nom, email et réseaux sociaux
- Utilisation de `Stack`, `Container`, `CircleAvatar`, `Text`
- Architecture modulaire avec des widgets séparés

Ce premier exercice introduit la composition de widgets **stateless** et la mise en forme graphique.

---

## Exercice 2 — Quizz interactif

Développement d’un mini-quizz avec gestion d’état :

- Affichage d’une question + image
- Plusieurs options de réponse
- Coloration dynamique après clic (vert = correct, rouge = incorrect)
- Bouton **Next** pour passer à la question suivante
- Score final + récapitulatif des réponses
- Utilisation d’un **StatefulWidget**, de `setState()` et de widgets personnalisés

Cet exercice introduit les interfaces dynamiques et la logique de contrôle dans Flutter.
